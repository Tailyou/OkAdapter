## 一、概述
通用Adapter，适用于单布局ListView、多布局ListView、单布局RecyclerView，多布局RecyclerView，带章节标题RecyclerView，GridView与ViewPage结合使用等情况。
## 二、版本
已在多个项目中使用，且已上传jCenter，最新版本0.0.6，直接在gradle中添加即可。
compile 'com.hengda.zwf:OkAdapter:0.0.1'
